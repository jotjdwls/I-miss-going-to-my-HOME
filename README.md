### Introduce
* 주제: 우리 집을 찾아라!
* 본가 동네를 모델링하여 구현

### skills
* Texture Mapping
* Phong Shading
* Perspective Viewing
* Collision → alert massage
* 'Moving' Obstacles

### User Interface
* '상' 이동

  * Up Key
  * Up Button '▲'
* '하' 이동
  * S, s
  * Down Key

* '좌' 이동
  * A, a
  * Left Key
  * Left Button '◀'
* '우' 이동
  * D, d
  * Right Key
  * Right Button '▶'
